Rapidpro v7 on Kubernetes
=================================

### Pre-requisites

1. A Kubernetes cluster

2. A container registry


### Steps:

To run Rapidpro on Kubernetes:

1. Build images

2. Push images to a container registry

3. Update Rapidpro helm values file

4. Helm install
